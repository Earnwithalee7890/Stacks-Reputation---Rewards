# CHANGELOG — OrdIndex

## [2.0.0] — 2026-02-19

### Added
- **Collection Layer**: `create-collection` and associated data structures.
- **Supply Tracking**: `total-supply` auto-updates on registration/removal.
- **Admin Control**: `remove-ordinal` for curation and moderation.
- **SDK**: TypeScript client for managing collections and ordinals.
- **Linking**: Optional linking of ordinals to collections.

### Changed
- **Registration**: Now accepts optional `collection-id` parameter.
- **Storage**: Optimized map structure for linking.

### Added
